# G-Scores

A full-stack web application for analyzing Vietnam National High School Exam scores.

---

## Tech Stack

### Frontend

- React
- TypeScript
- Vite

### Backend

- NestJS
- Prisma ORM

### Database

- PostgreSQL
- Supabase

### Infrastructure

- Docker
- Docker Compose

---

## Project Structure

```text
apps/
    api/
    web/

docs/

infra/
```

---

## Features

- Search student by registration number
- Statistics by subject
- Top 10 students in Group A
- CSV import into PostgreSQL
